# Sirius Project 🌌

Welcome to the **Sirius Project**! This repository holds a very special place for me because it represents the absolute beginning of my software development journey. It is the **very first React website** I ever built when I was a complete amateur and discovering code for the first time.

Instead of refactoring or cleaning up the code to make it look "perfect," I decided to upload it **exactly as it was written on day one**. It stands as a digital archive of my root foundations, my initial learning curve, and a benchmark to look back on and see how far I have come.

---

## 🛠️ Tech Stack & Structure (As of Day 1)

* **Library:** React (Legacy version, configured via older CRA templates)
* **Routing:** `react-router-dom` (v5 styling)
* **Styling:** Custom CSS (`App.css`, `index.css`)

---

## 🎯 What This Project Reflects

Looking back at this codebase today with a modern engineering mindset, I can point out all the classic "first-timer" patterns that I have since outgrown:
- **Local Character Overreliance:** Using Turkish characters in file naming, components, and routing paths (e.g., `Hakkımızda.js` or `path="/üyelik"`), which is a wonderful teaching lesson on why production environments favor standard English alphanumeric routing.
- **Component Tag Cascading:** Using lowercase tags like `<switch>` instead of React's upper-case `<Switch>` component wrapper, showing my early steps into JSX parsing quirks.
- **Global CSS Overrides:** Utilizing heavy universal selectors (`*`) for font-family defaults directly within standard stylesheets rather than leveraging root body inheritance in `index.css`.

**I kept all of these exactly the same** because hiding your early mistakes also means hiding your progress.
